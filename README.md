# mongo2neo4j
MongoDB Mongoose to Neo4j Importer

## Installation

### Install Neo4j

see [Neo4j](https://neo4j.com/)

### Install SemSpect

see [SemSpect](https://www.semspect.de/)

### Install Python

see [Python](https://python.org/)

### Install Python libs

```
# pip install pymongo
# pip install neo4j
```
